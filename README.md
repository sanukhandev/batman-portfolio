# 🦇 Cinematic Batman Scroll Portfolio

A highly cinematic and premium scrollytelling personal portfolio website inspired by the dark, mysterious, and heroic aesthetics of Batman.

## ✨ Features

- **Theatrical Scrollytelling**: A scroll-driven animation experience with layered text reveals.
- **Iconic Climax**: Culminates in a striking "I AM BATMAN" moment.
- **Custom Animated Logo**: Features a custom SVG logo animation.
- **Premium Typography**: Refined typography utilizing `Inter` and `Bebas Neue` for maximum impact and readability.
- **Dark Aesthetic**: A sleek, dark-mode-first design tailored for creative developers and designers.

## 🚀 Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- React
- Tailwind CSS
- Custom Animations

## 🛠️ Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Repository Setup

This project is configured to be pushed to its remote repository:
```bash
git remote add origin https://github.com/sanukhandev/batman-portfolio.git
git branch -M main
git push -u origin main
```
